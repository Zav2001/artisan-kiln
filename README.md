# The Artisan Kiln — Ceramic Tile Order Form

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Redux Toolkit
- Framer Motion
- Jest

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build    # production build
npm run start    # production server
npm test         # unit tests
```
